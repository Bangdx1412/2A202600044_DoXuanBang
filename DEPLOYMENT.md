## Render Setup
1. Push the repository to GitHub.
2. In Render Dashboard, choose `New -> Blueprint`.
3. Select this repository.
4. Set `Blueprint Path` to `06-lab-complete/render.yaml`.
5. Ensure `AGENT_API_KEY` exists in Render Environment.
6. Provide `OPENAI_API_KEY` if you want a real LLM instead of the mock response.

## Test Commands

### Health Check
```bash
curl https://ai-agent-production-hb3q.onrender.com/health
# Expected: {"status":"ok",...,"redis_connected":true}
```

### Ready Check
```bash
curl -i https://ai-agent-production-hb3q.onrender.com/ready
# Expected: HTTP 200 and {"ready":true,...}
```

### Authentication Required Check
```bash
curl -X POST https://ai-agent-production-hb3q.onrender.com/ask \
  -H "Content-Type: application/json" \
  -d '{"user_id":"test-user","question":"Hello"}'
# Expected: 401 Unauthorized
```

### Authenticated API Test
```bash
curl -X POST https://ai-agent-production-hb3q.onrender.com/ask \
  -H "X-API-Key: YOUR_CURRENT_AGENT_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"user_id":"test-user","question":"Hello"}'
# Expected: 200 OK
```

## Notes

- If `AGENT_API_KEY` was auto-generated by Render and you do not know it, reset it in Render Dashboard before running the authenticated test.
